Landing page for howtobebetter.ai — a newsletter about using AI for personal growth.

## Setup

1. Replace Kit form placeholders in `index.html` (search for `REPLACE_WITH`)
2. Custom domain: `howtobebetter.ai` configured via CNAME + DNS
